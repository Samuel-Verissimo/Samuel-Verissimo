## Sobre mim

Sou **Senior Software Engineer**, com experiência no desenvolvimento de aplicações web e mobile, arquitetura de sistemas, integrações complexas, microserviços, soluções com IA e produtos SaaS.

Atuo de ponta a ponta no desenvolvimento de software, passando por **backend, frontend, mobile, infraestrutura, cloud, observabilidade e integrações com sistemas externos**. Tenho experiência em ambientes de alta criticidade, evolução de produtos digitais, fluxos de pagamento, automação de processos e construção de soluções escaláveis, seguras e manuteníveis.

Também contribuo com decisões técnicas, definição de arquitetura, code reviews, padronização de código e apoio na evolução técnica de times.
